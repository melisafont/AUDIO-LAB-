# AUDIO-LAB-
AudioLab es una web interactiva diseñada para la captura, procesamiento y análisis de audio en tiempo real. Permite grabar audio desde el micrófono del usuario, visualizar formas de onda y espectros de frecuencia, aplicar procesos de remuestreo y reducción de profundidad de bits, y exportar los resultados en formatos de audio estándar.
